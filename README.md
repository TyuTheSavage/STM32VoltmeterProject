# STM32 Voltmeter Project

### Hardware Specifications
* **MCU:** STM32 F091RC (ARM Cortex-M0)
* **Display:** ST7565R (128x64 Monochrome LCD)
* **Range:** 0V - 3.3V (Direct ADC Input)

### Environment
* **RTOS:** FreeRTOS
* **Language:** C
* **Toolchain:** STM32CubeIDE / GCC
